敲碎传统教育的蛋壳

发表于 2017-03-22 |

写在前面

动笔开始写个人博客是早就有了的想法，但无奈一是自己太懒，二是学业繁忙，故虽然早早的建起了个人博客，但是写文章的事却在日程表上一拖再拖。前段时间和璇聊天，无意中提到一种观点，“如果看完一部电影，却没有任何的文字记载，便相当于没看”，黯然惶恐。恰好这段时间思维跳动以致常常失眠，若这些文字不能得到记载，这段时间的眠岂不是白失了？第一篇博文，练笔之作，权当站在人工智能的元年对这段时间教育领域的一点思考和看到的一些报道作一个总结。

传统教育的蛋壳

从2012年，中国互联网的崛起，但现在为止，生活的众多方面都发生了颠覆性的变化，在线教育的概念也一炒再炒，但实话实说，学校教育时至今日，还是太行与王屋，岿然未动。曾经在大学英语课上学过一篇戏谑的文章，里面一句话让我印象深刻：

若柏拉图来到现代，他必定会讶异于现代社会天翻地覆的变化，而当他来到学校，则会长舒一口气，’还是老样子‘。

中国的教育有病吗？其实不好说，拜读郑也夫老师的《吾国教育病理学》，郑老师从素质教育论谈起，到教育行政的专权，为中国教育把脉问诊，大悟不少，但是我还是不能断然肯定中国教育大病入骨，是否教育的从业者就应该奋不顾身的去改变中国当下的教育。何况高考的公平性，中国基础教育的扎实性（尤其数学学科）是否真的值得诟病？

一直出国之后，被一些中国媒体蒙住的双眼才渐渐能够看得明白事情，号称素质与兴趣为王的西方教育也不再是报道里面对中国教育铿锵有力的控诉，而是一个个活生生的人，我们每天和他们交流，一起学习，被放在同样的平台下比较，中西教育，孰优孰劣，我想我应该亲眼见到了一个很好的比较说明。

  • 其一，外国学生往往只有很少一部分人带眼镜，但是我周围的中国男生，不戴眼镜者，凤毛麟角；

  • 其二，外国学生健身成风，中国学生体质明显偏弱，身形较为瘦小；

  • 其三，学术表现上，中外学生往往平分秋色，但有一个有意思的现象，虽然中国学生人数众多（20%），但助教与老师中，中国人或者华人所占比例极低；

  • 其四，西方兴趣导向有，但是更多人其实也是就业导向，与就“易”导向，此“易”就是容易的易，意思也是字面的意思；

  • 其五，西方本科教育质量极高，有些课程和本科生同上，国外本科生在学业上的表现不让中国研究生分毫；

国外生活对中国学生来说颇为不易，tough journey名不虚传，这种和外国学生的比较也表现在方方面面，巨细难以陈述，每个方面究其根底，都不啻于一篇毕业论文的体量，但我想以上五点仍然能够浅显表明中国教育的有些问题：

  • 高考之前承受的学习压力过于巨大，对学生的身体与精神是极大的摧残；

  • 如果说中国基础教育就内容上可算及格，那么中国的高等教育，病入膏肓；

  • 这个算是我一个本人的感受，中国地大人多，教育优质资源往往分配极为不均，各省超级中学林立，我见过太多的聪明的同学，因为环境与老师的原因，无法将他们的聪明才智发挥出来；

这些问题，积重难返，教育关系民生，整个教育系统又尾大不掉，加上中国传统的尊师重教的文化氛围，政府，学校，老师，家长，组成了一个统一阵线联盟，学生犹如鸡仔，在这统一阵线数千年来构筑的蛋壳之下安然长大，而中国的教育学人又将希望寄托在管理者上，希望能够借自己的研究，为教育找到好的模式，推广开来造福百姓，于是我们看到了近些年的教改，这改那改，高考不改，不能否认少部分改革，但大部分改革还是行的换汤不换药的买卖。但巧得是，分配资源这些事情，恰恰又是互联网的长项，在线教育应运而生，也就不值得奇怪，几年过去，资本冷静下来，在线教育的企业成批倒下，存活下来的有几家企业也让人看到遥远的希望，不由得让我有了一种谜一样的自信，传统教育的蛋壳有朝一日，终会破裂，无论多么稳固的联盟也会伴随着这一过程而瓦解，而这个体系之下的学校，老师，家长，都会拥抱变化。AI与互联网帮助下的教育，将会变得高效，资源分配也会变得均衡，从而将学生从繁重的压力之中解救出来，而所谓的超级中学也将有朝一日不复存在。

新教育

其实这一段儿才是我的老本行，上面都是一个门外汉的胡言乱语。记得当年，老师提到了教育的误区，其中之一就是，很多人以为教育是龙头，舞动龙头，龙身也跟着起舞，其实从历史的进程看，教育一直是龙尾，只有当科技兴起，政治清平，才能带动教育的生机。而现在的契机，就是人工智能的兴起，而传统教育的痛点也会被人与智能机器的合作，一一击破；下面有几个教育可能发生根本改变的点，而且均是现在切实可行的方案，另外，脱离场景而谈应用，其实都是耍流氓。

因材施教–自适应教育

场景：我记得大三，帮璇写关于儒家的一篇论文，里面提到了一个非常有意思的小故事：

子路问:“闻斯行诸?”

子曰:“有父兄在，如之何其闻斯行之?”
冉有问:“闻斯行诸?”
子曰:“闻斯行之。”

公西华曰:“由也问，闻斯行诸？子曰，‘有父兄在’;求也问闻斯行诸，子曰‘闻斯行之’。赤也惑，敢问。”

子曰:“求也退，故进之;由也兼人，故退之。”

因材施教，是教育多少年来的梦想，孔子还说过：“不患人之不己知，患不知人也。”如何知人？1个班主任带60多个学生的时代可能么？即便班主任学究天人，才通古今，比之孔夫子犹有胜之，但是大班教学的形式已然决定了因材施教的不可行。

解决方案：现有的题库类软件给出了一条可行的思路，留住学生在我们的平台，学生不断的做题就会不断的有数据产生。利用这些数据，设定因子，调整算法，便能建立学习模型，针对每个特定的人而推荐截然不同的内容。不同的学生有不同的知识薄弱点，也有知识的盲点，这些在算法面前一目了然。于是针对这些知识弱点，就可以给予特定辅导和特定的题目，这就是因材施教。

及时反馈–人工智能助手

场景： 你对一个题冥思苦想了很久，却苦于没有一个人为你答疑解惑，又或者是你希望自学乐器或者绘画，却乏良师指导学习中的一些错误。如果上面这几种体验情况比较少的话，那一定都体验过，自己想去学习，看见大段的文字和各种公式不厌其烦，于是会想，如果有个人能手把手的教我做这个就好了。

解决方案：各种搜题类的app给了一个比较典型的解决方案，你可以拿出手机扫题目，然后通过图像识别与分析，可以在数据库中迅速的定位到该题所属的知识脉络，不仅仅是给使用者提供解题答案，更是帮学生来完成对这个知识点的补足。当然如果你愿意说几句话，描述一下就更棒了，通过语义分析，更能直接完成对该题所属知识脉络的清理，过滤掉你可能已经掌握的部分，集中处理那些你一知半解的盲点。

音乐和绘画的自学也是同一个道理，音乐自学借助语音识别，能够给出音阶的正确与错误反馈，还能控制节奏，帮助练习。

决策–专家系统

场景：当你面临毕业，一片茫然，读研，工作，保资或者出国？每条路都仿佛对应了不同的结果，事实也是这样，你想要借鉴别人的经验，发现别人上了四年大学，你被大学上了四年。和你境遇相似的呢，回家工作，你又总认为还有更好的出路。于是，你拍案而起：“出特么的国”，决定既定，你心中大安，为之四顾，为之踌躇满志。但实际一看，国外学校那么多，自己的条件如何匹配？

解决方案：上面的场景其实有两个：

  • 其一，毕业决策，这一场景的决策数据来源我还没想到一个万全的办法，毕竟涉及过多隐私，从社交媒体的挖掘与分析尚需探讨，但是只要有数据，就能够为个人做出相应的推荐，至少能够为各种潜在的方向做出评级（猛然有一丝黑镜的味道）。
  • 其二，留学决策，这个里面有太多的留学中介，留学中介手头就有很多学生的资料，从学业成绩到留学毕业之后的去向，不一而足，数据不是难事，学习这些资料，做出相应的推荐，相信不会太难。

小结

一些简单的不成熟的想法，一篇概括性的文章，其实每个场景与功能都包容万象，值得大书特书，这篇文章也相当一个备份的作用罢，提醒以后，尚有许多内容等我来填充。

我和教育

  • 2011年，高三，立志，关注教育；
  • 2012年，高考，报考北师大，不服从专业调剂被退回；
  • 同年，进入北理工计算机专业；
  • 2013年，谋求转专业，无教育学而半途告结；
  • 同年，闲暇时间支教一年，暑假赴贵州支教两月；
  • 2014年，入读北理教育学双学位；
  • 2015年，互联网崛起，对传统教育学失望，希望以科技载道；
  • 2016年，毕业，出国留学至今以求拓宽视野，不囿于媒体成见；
