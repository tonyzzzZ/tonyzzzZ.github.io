围绕题库打造教育生态链

发表于 2017-03-22

小米的商业模式

关注了一个投资人的专栏，记得里面提到了一个观点，近十年来，见过的最值得关注的商业模式，一是ofo，摩拜这些共享单车的商业模式，二就是小米的商业模式。

每当提到小米，总是绕不开价格，性价比这些词，小米从性价比打起，迅速的攻城略地，成为如今中国未上市公司中估值第二的公司。很多人还停留在小米的便宜的印象流的基础上，但是登上小米的官网一看就能知道，小米现如今除了手机业务，已经围绕着手机形成了一个相当丰富的产业链。智能硬件围绕着小米手机，迅速的拓宽了小米的可能性。

在智能硬件方面，小米的策略是用生态链的打法，端到端全部解决。

“设备公司也是我参股和支持的，系统和模组也是我做的，很容易跟手机连接在一起，当这样的设备越来越多的时候，小米手机生态链的优势会体现出来。两年半下来，我们可连接的设备数已经变成了千万。”雷军说，“我们希望拥有更多的合作伙伴，但是在初期的时候，大家不愿意跟我们连的时候，我们一定得做一个样板功能，如果你没有样板谁连你呢？这个设备是不是可用，还有未来标准的可用性，最重要还是谁拥有的用户量越大，标准就会变成事实上的工业标准。”

就像雷军所说，“谁的用户大，谁的标准就会成为事实上的工业标准”，标准一旦既定，用户与投资也就都来了。

题库的生态链

生态链的打法在教育领域也是绝对行之有效的。

先不谈生态链，举一个小小的例子，猿题库算是国内领头的题库类产品，打开猿题库做里面的试卷套题，会发现下面的两个问题：

  • 填空题，解答题这类题没有办法线上答题，所谓的自适应学习的数据来源大多数来自于选择题。而有的老师曾经提到过，他怀疑自适应学习的有效性，因为他见过太多的学生把题库类产品点开，随便点了玩儿。
  • 公式的显示还是用的图片显示，其实如果使用 mathJax 这种支持网页显示公式的方式，能够有更好的视觉效果不说，数据上与反馈上都将带来意想不到的效果。

如果说再移动互联网时代，流量最值钱，在巨头林立的移动互联网时代，内容最值钱，那么在人工智能时代，最值钱的一定是数据，猿题库在不能收集学生的填空题和解答题这一项上，就流失了多少数据啊。

此外，掣肘题库类产品发展的因素有很多，其中很重要的两点，我认为一是电子产品本身的不纯粹性（可以玩游戏，刷微博）；二是效率仍然不高，特别是书写公式上，比起用纸和笔，实在差太多，尤其是对于手机，公式的输入近乎无解。

回到生态链，教育里的场景之多，是难以想象的，围绕题库打造一个生态链，是一个很好地突破点，比如说著名的Bamboo Spark
，可更换纸张，即时的将内容反馈到电子屏幕，这样的设计，能不能解决公式的输入，无论是物理的，化学的还有数学的？当图像识别的准确率达到一定程度的时候，我相信是可以的。当然，第一步仍然是停止用图片显示公式，转而使用标记语言，使得公式即便输错了还有可以修改的余地。一旦题库类用户积累到一定程度，这种产品完全能够将题库类产品升华到另一种层次,下面是一个视频，能翻墙的可以看看：

结语

可以解决掉题库类产品缺点的科技其实有很多，将思维局限在小小的一部手机上面，我觉得是不理智的，也不符合时代的进步，移动互联网终究会是人类科技中的一个步骤，围绕核心价值打造一个可以商用又能改革教育的模式，才是正道。
